Bytecode virtual machine (port) based on the one in "Crafting Interpreters".

*Info*
- Single-pass compiler (no AST)
- Stack-based
- Mark-and-sweep
- Upvalue management
- String interning

*Constructs*
- Variables
- Data types
- Operators
- Control flow
- Functions
- Closures
- Classes
- Inheritance
- Native functions

More examples in tests in main.odin, but the way the tests are currently setup is terrible.
